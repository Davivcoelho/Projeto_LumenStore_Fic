#  Lumen Store – Dashboard Comercial (Power BI + Python)

Este projeto consiste no desenvolvimento de um dashboard gerencial para análise do desempenho comercial de uma empresa varejista fictícia (Lumen Store).

A solução combina Power BI para visualização executiva e Python (Google Colab) para análises complementares e tratamento de dados.

##  Objetivo do Projeto

Construir uma solução de Business Intelligence capaz de transformar dados operacionais em informações estratégicas para apoio à tomada de decisão.

O dashboard permite analisar:

- Evolução do faturamento ao longo do tempo
- Impacto dos descontos na receita líquida
- Performance de vendas por filial
- Concessão de descontos por vendedor
- Participação das famílias de produtos no faturamento
- Top 5 produtos por família (faturamento e lucro)
- Afinidade de compra entre famílias de produtos

##  Principais Indicadores

O dashboard apresenta:

- Faturamento Bruto
- Total de Descontos
- Receita Líquida
- Lucro Total
- Evolução mensal do faturamento
- Ranking de vendedores
- Performance por filial
- Análise de famílias de produtos
- Matriz de afinidade entre categorias

O lucro foi calculado como:

Quantidade Vendida × (Preço Unitário - Custo Unitário)


## Tecnologias Utilizadas

- Power BI
- DAX (Data Analysis Expressions)
- Python (Google Colab)
- Pandas
- Modelagem de dados (modelo estrela)
- Análise temporal (Ano-Mês)
- Lógica de ranking dinâmica (Top N por grupo)
- Uso de Inteligência Artificial como apoio à programação em Python e estruturação de análises
